# Submission Packages

## Purpose

This directory contains venue-specific submission bundles derived from the scientific master draft.

## Available Packages

- `journal_of_cheminformatics/`
  - target article type: `Methodology`
  - best fit when the paper is framed as benchmark + methodology + feature insight
- `bmc_bioinformatics/`
  - target article type: `Research article`
  - best fit when the paper is framed as bioinformatics benchmark + controlled evaluation study

## Common File Pattern

Each venue folder contains:

- journal-specific main manuscript `.tex/.pdf`
- journal-specific supplementary-information `.tex/.pdf`
- checklist
- cover-letter template
- title-page template
- declarations template
- submission manifest
- venue-specific conversion notes

## Current Status

Both venue bundles are compilable and already include:

- author name
- affiliation
- funding status
- acknowledgements status
- repository URL

Remaining submission metadata is limited to:

- corresponding-author designation
- contact email
- any optional postal/contact block required by the target journal

Use the venue README inside each folder to see remaining blockers before upload.
