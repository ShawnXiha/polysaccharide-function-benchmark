# Submission Packages

This directory contains venue-specific submission packages prepared from the current polysaccharides KG manuscript.

## Packages

- `bmc_bioinformatics/`
  - Recommended article type: `Research article`
  - Rationale: the manuscript contains a database/KG resource component, but it also includes a standalone benchmark and methodological analysis. BMC's own database-article guidance says that database descriptions with publication-worthy research should either be submitted as a research article or split into separate database and research papers.

- `database/`
  - Recommended framing: `Database/resource article`
  - Rationale: the manuscript already fits the journal's focus on biological databases, database tools, ontology resources, and biocuration-oriented methodology.

## Shared manual items still required before submission

- Final author list, affiliations, and corresponding author details
- ORCID iD for the submitting/corresponding author
- Public repository or landing page URL for the KG assets
- Final funding information
- Final conflict-of-interest confirmation
- Final suggested/excluded reviewer list
