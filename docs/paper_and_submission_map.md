# Paper and Submission Map

## Goal

This document gives a short map from the scientific master draft to the venue-specific submission bundles.

## Master Scientific Draft

- manuscript:
  - `paper_poly_core_v1.tex`
  - `paper_poly_core_v1.pdf`
- bibliography:
  - `references_poly_core_v1.bib`
- latest self-review:
  - `paper_review_selfcheck_round4.md`

Use the master draft when:

- changing the scientific story
- revising tables or figures
- updating claims after new experiments

## Which Version To Edit

Edit the master draft first when the change affects:

- the core scientific claim
- tables, figures, numbers, or citations
- benchmark or evaluation interpretation

Edit the venue draft directly when the change affects:

- title-page metadata
- abstract formatting
- declarations
- journal house style
- upload-package structure
