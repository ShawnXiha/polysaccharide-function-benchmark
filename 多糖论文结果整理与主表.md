# 多糖论文结果整理与主表

## 任务定义

- 任务：基于 `DoLPHiN` 监督记录，进行天然多糖的多标签结构-功能预测
- 主数据集：`data_processed/dataset_publishable_supervised_v1.jsonl`
- 规模：`4121` 条记录，`18` 个功能标签
- 主评测协议：`random_split`
- 主指标：`macro-F1`
- 次指标：`exact match ratio`

## 论文可写结论

### 可以强写的结论

1. 在当前多糖监督基准上，简单迁移的 glycan-style sparse baseline 仍有明显提升空间。
2. 多糖特异结构表示在 `macro-F1` 上给出一个有限且 split-sensitive 的提升，特别是 `MW + branching + residue` 三类结构 token。
3. 通过控制消融，`MW` 和 `residue` 是最强贡献项，`branching` 提供稳定次级增益。
4. benchmark 支持多个有竞争力的 sparse baseline；`TF-IDF + logistic` 和 `Linear SVM` 在 `exact match` 上更强，说明指标选择会改变简单模型的排序。

### 不能强写的结论

1. 不能把当前结果写成强 `cross-source biological generalization` 结论。
2. 不能把当前结果写成强 `evidence-aware modeling` 结论。
3. 不能把 `CSDB` 结果当作主监督主张的核心证据。
